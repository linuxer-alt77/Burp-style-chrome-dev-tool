# CSS Directory

This directory contains all stylesheets for the HTTP Request Repeater extension.

## Planned Stylesheets

- `panel.css` - Main panel styling
- `themes/dark.css` - Dark theme colors
- `themes/light.css` - Light theme colors
- `components/request-list.css` - Request list component styles
- `components/editor.css` - Request editor styles
- `components/response-viewer.css` - Response viewer styles
