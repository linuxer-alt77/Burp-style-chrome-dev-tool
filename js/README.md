# JavaScript Modules Directory

This directory contains all JavaScript modules for the HTTP Request Repeater extension.

## Planned Modules

### Core

- `background.js` - Service worker for background operations
- `devtools.js` - DevTools page initialization
- `panel.js` - Main panel controller

### Modules

- `network-capture.js` - Network request capture logic
- `request-manager.js` - Request storage and management
- `request-editor.js` - Request editing functionality
- `request-sender.js` - HTTP request sending logic
- `response-handler.js` - Response parsing and display
- `message-handler.js` - Message passing between components
- `storage-manager.js` - Chrome storage operations
- `ui-controller.js` - UI state management
