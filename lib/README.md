# Third-Party Libraries Directory

This directory contains third-party libraries used by the extension.

## Planned Libraries

- **Monaco Editor** (or CodeMirror) - Code editor for request/response display
- **highlight.js** - Syntax highlighting for various content types
- **DOMPurify** - XSS sanitization for displaying responses
- **json-formatter** - JSON prettification and formatting

## Usage

Add libraries here to avoid CDN dependencies and ensure offline functionality.
